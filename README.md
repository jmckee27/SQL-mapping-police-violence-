# SQL-mapping-police-violence-

A relational database to analyze Black Lives Matter era police violence

## Description

This relational database within MySQL is a comprehensive record of 10,000+ violent police incidents taking place from 2018-2021. This database however analyzes the incidents taking place specifically from 2018-2020 to cross-reference their relevance against the Black Lives Matter Movement. Queries to isiolate the law enforcement agencies involved, subsequent officer charges, black victims, victim/officer, and bodycam utilization are implemented
